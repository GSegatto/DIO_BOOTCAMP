# DIO_BOOTCAMP
Repositório com projetos, desafios e anotações do Bootcamp DIO de Inteligência Artificial, Dados e Cibersegurança. Inclui estudos e aplicações práticas de IA Generativa, Prompt Engineering, Excel, SQL, Python, análise e visualização de dados, automação e segurança digital, reunindo minha evolução ao longo da trilha.
