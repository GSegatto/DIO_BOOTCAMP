# DIO_BOOTCAMP
Repositório com projetos, desafios e anotações do Bootcamp DIO de Inteligência Artificial, Dados e Cibersegurança. Inclui estudos e aplicações práticas de IA Generativa, Prompt Engineering, Excel, SQL, Python, análise e visualização de dados, automação e segurança digital, reunindo minha evolução ao longo da trilha.

## Objetivo

Este projeto foi desenvolvido como parte de um desafio da DIO, utilizando o NotebookLM como ferramenta de apoio aos estudos.

O objetivo foi aprofundar os conhecimentos em Engenharia de Prompts, compreendendo como elaborar instruções mais claras e eficientes para obter respostas de maior qualidade em modelos de Inteligência Artificial. Durante o processo, foram documentados conceitos, testes, exemplos práticos e boas práticas para facilitar futuras consultas.

Além do aprendizado técnico, este projeto demonstra a aplicação de IA como ferramenta de organização do conhecimento e apoio ao desenvolvimento profissional.

## Tema Escolhido

# Engenharia de Prompts para Grandes Modelos de Linguagem (LLMs)

### Objetivos específicos
Compreender como os prompts influenciam as respostas dos modelos de IA;
Aprender técnicas de prompting, como:
- Zero-shot;
- One-shot;
- Few-shot;
- Chain of Thought (CoT);
- Criar prompts reutilizáveis para diferentes cenários;
- Organizar um material de estudo para revisão e consulta.
